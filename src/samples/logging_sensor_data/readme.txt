■サンプル名：
logging_sensor_data

■サンプル概要：
LSM330 から取得される加速度と角速度情報をMicro SDカードに保存します。

■サンプル説明：
データが取得された時刻情報、その時の制御値（モーターへのDuty比）、加速度（3軸）、角速度（3軸）をMicro SDカードに保存します。表現形式は16進数のCSVです。

時刻情報, 制御値, 加速度X, 加速度Y, 加速度Z, 角速度X, 角速度Y, 角速度Z

の並びで保存されます。

■実行方法
1．プログラムを制御基板に書き込む
2．電源を入れる
3. SW1 を押して制御を開始する

■注意点：
Micro SDカードへの書込処理中に電源を落とさないようにしてください。
SW1を押下後、走行中はMicro SDカードへの書き込みが断続的に発生します。再度SW1を押下してモータの回転を止めると書込み動作が止まる為安全に電源を切る事が可能です。
