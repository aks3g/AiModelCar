■サンプル名：
using_physical_conversion

■サンプル概要：
LSM330 から取得される加速度と角速度情報を基にして姿勢の推定を行う

■サンプル説明：
センサーから得られた情報をもとに、右カーブ、左カーブ、登り、降り、走行安定性の判定を行います。

姿勢データはビットマップになっており、値はSDカードに保存されます。
タイムスタンプ、DUTY比、姿勢、センサデータ（加速度３軸、ジャイロ３軸）の順に並びます。

■実行方法
1．プログラムを制御基板に書き込む
2．電源を入れる
3. SW1 を押して制御を開始する

■注意点：
Micro SDカードへの書込処理中に電源を落とさないようにしてください。
SW1を押下後、走行中はMicro SDカードへの書き込みが断続的に発生します。再度SW1を押下してモータの回転を止めると書込み動作が止まる為安全に電源を切る事が可能です。
